Three ways to add CSS file in HTML.

There are three ways to how to add CSS file in HTML:-

  1. Inline CSS
    2. This used in a HTML page using style attributes.
  2. Internal CSS
    3. It means to add a style tag and place the HTML page's style with in it.
  3. External CSS
    4. It means to create .css file and place all your style inside that .css file.
